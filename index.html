风格说明

这套博客系统基于 GitHub Pages，默认可直接托管到 `/username.github.io`。 整体视觉参考 Apple 风格：留白、圆角、纤细边界与克制的蓝色强调色； 内容节奏以代码友好字体与单栏排版为主，适合程序员写技术文章、教程与复盘。

发布方式

  1. 在 data/posts.json 新增一篇文章元信息（标题、摘要、日期、标签）。
  2. 在 posts/slug-name.md 新建正文。
  3. 推送到 GitHub Pages 分支即可自动生效。
