很多人做技术博客时会陷入“花里胡哨”，但真正长久的风格来自**一致性**。

## 核心规则

### 1. 强制留白

页面在视觉上尽量有呼吸空间。让阅读更放松，不要到处塞装饰。

### 2. 减少色彩种类

主色 + 中性色就够了。

### 3. 使用语义化字体

标题、正文、代码要区分清楚。

> 你看到的每一个文字层级都应该传达信息等级，不要靠更大字号堆砌。

### 4. 程序员语境的边界感

细线框、网格、简洁阴影和小圆角，能让内容更像工程化系统。

```css
:root {
  --bg: #f5f5f7;
  --accent: #0071e3;
  --text: #111318;
}
```

## 结尾

程序员审美不是冷冰冰，而是“有秩序的清爽”。
